Pixel Password is a visual authentication platform I invented and built end-to-end.

**🔗 Live Site:** [https://pixelpassword.com](https://www.pixelpassword.com/)

---

## 📌 Overview

Pixel Password enables secure, image-based sign-in through an interaction model I developed, supporting workflows such as paste, drag-and-drop, and upload.

- Invented the concept and interaction model
- Built the product using React, Next.js, and TypeScript, Tailwind CSS
- Refined features based on real usage and feedback
- Used AI-assisted workflows to accelerate development
